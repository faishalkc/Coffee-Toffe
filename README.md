# Coffee Toffee Landing Page

Coffee Toffee Landing Page is a responsive landing page website that showcases information about Coffee Toffee, including brand highlights, featured menu offerings, company profile, and contact information.

Built using HTML, CSS, and Bootstrap, this project focuses on delivering a clean, modern, and user-friendly browsing experience across desktop and mobile devices.

2. Open the project folder
