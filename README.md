# Coffee Toffee

Coffee Toffee is a responsive landing page website that showcases information about Coffee Toffee, including brand highlights, featured menu offerings, company profile, and contact information.

Built using HTML, CSS, and Bootstrap, this project focuses on delivering a clean, modern, and user-friendly browsing experience across desktop and mobile devices.

## Features

* Responsive landing page design
* Modern and clean user interface
* Hero section
* About section
* Featured menu section
* Brand information
* Contact information
* Mobile-friendly layout

## Technologies Used

* HTML5
* CSS3
* Bootstrap

## Installation

1. Clone this repository

```bash
git clone https://github.com/faishalkc/Coffee-Toffee.git
```

2. Open the project folder

3. Run `index.html` in your browser
